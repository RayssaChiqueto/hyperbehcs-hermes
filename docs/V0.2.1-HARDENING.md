# v0.2.1 Doc / CI Hardening

v0.2.1 hardens the public v0.2 seed so a normal user can clone, install, verify, and inspect the public descriptor surface without local Acer/Asolaria runtime access.

## Added hardening

- README maturity label now reflects the strict-authority public release line.
- Package metadata is bumped to `0.2.1`.
- CI installs the package with `python -m pip install .`.
- CI smoke-tests the console command with `hyperbehcs-hermes --help`.
- CI verifies `examples/packet.hbp`.
- CI verifies `examples/authority_surface.hbp`.
- CI runs append-only chain verification on `examples/authority_surface.hbp`.
- CI verifies every `packs/*/*.hbp` descriptor pack.
- CI runs append-only chain verification on every descriptor pack.
- CI smoke-tests `list-authority` and `list-packs`.
- CI verifies approved promotion example succeeds.
- CI verifies revoked promotion example fails as expected.
- CI continues enforcing no tracked JSON hot path.
- CI continues enforcing public-safe content needles.

## Public crosswalk seed

v0.2.1 also adds a public-safe local proof crosswalk:

```text
docs/LOCAL-PROOF-CROSSWALK.md
packs/local/public_local_crosswalk_descriptors.hbp
packs/local/public_local_crosswalk_descriptors.hbi
packs/local/public_local_crosswalk_descriptors.sha256
packs/local/public_local_crosswalk_descriptors.hex
```

The crosswalk maps local Acer/Asolaria operational classes into public descriptor rows, without exporting private runtime, auth, leases, inbox/outbox data, raw local files, or live daemon/control surfaces.
